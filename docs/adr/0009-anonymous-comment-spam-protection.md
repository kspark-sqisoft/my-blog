# ADR-0009: 익명 Comment 스팸 방지로 throttler + 길이 제한 채택

## 상태 (Status)

Accepted - 2026-06-02

## 컨텍스트 (Context)

Comment는 로그인 없는 익명 독자가 작성한다(PRD). 인증이 없으므로 대량 자동 등록·도배 등 남용에 노출된다. NF1(보안)은 Comment 입력의 안전성과 스팸 억제를 요구한다. 과한 방어(예: 캡차·가입 강제)는 독자 참여라는 제품 목표와 충돌한다.

평가 기준: 남용 억제 효과, 독자 마찰 최소화, 구현 비용.

## 결정 (Decision)

익명 Comment 작성 보호로 두 가지를 채택한다: (1) `@nestjs/throttler`로 Comment POST에 레이트 리밋, (2) `body` 최대 길이 상한 검증. 가입·캡차는 도입하지 않는다.

## 결과 (Consequences)

긍정:
- 로그인 없이도 기본적인 도배·대량 등록을 억제한다.
- 독자 참여 마찰을 최소화한다.

부정/감수해야 할 것:
- 정교한 분산 스팸(다수 IP)은 막지 못한다.
- 레이트 리밋 임계값과 길이 상한값 튜닝이 필요하다(상한값은 오픈 이슈).

## 검토 시점

스팸이 임계값을 우회해 실제 문제가 되면 캡차·모더레이션 등 추가 방어를 새 ADR로 재평가한다.
