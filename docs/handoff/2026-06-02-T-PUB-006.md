## 가드 배선 해결 (T-AUTH-004 선결 과제)

`JwtAuthGuard`(AuthGuard('jwt'))는 passport 전역 레지스트리의 'jwt' 전략을 사용한다.
PublishingModule이 AuthModule을 import하면 JwtStrategy가 인스턴스화되어 가드가 동작.
→ 다른 보호 모듈(E4 Conversation 등)도 동일하게 AuthModule을 import 하면 됨.
