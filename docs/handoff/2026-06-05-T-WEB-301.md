## 결정한 것
- TipTap `Color` 확장(인라인 style)을 **사용하지 않음** — 우리 sanitize 화이트리스트가 인라인 style 을 금지하기 때문. 두 카테고리(색/크기)를 별도 mark 로 분리해 각각 span class 부착.
- `Video` 노드는 커스텀 — markdown-it 변환(T-INFRA-303) 결과의 `<video>` 와 같은 속성으로 직렬화. 카드(T-WEB-203)와 상세 (T-WEB-303 예정) 양쪽이 같은 노드 형식을 본다.
- 도구바의 색/크기 select 는 정렬된 프리셋만 — 임의 hex/px 입력은 ADR-0021 범위 외.
- 모듈 로드 시 화이트리스트 회귀 가드(throw) — 작성자 화면이 빈 화면이 되어 빠르게 인지.
