# Handoff — T-WEB-307 (에디터 툴바 아이콘 재디자인 + 서식 확장)

날짜: 2026-06-05 / feature: rich-text-editor

## 한 일
레퍼런스 툴바 스크린샷에 맞춰 글 에디터 툴바를 **아이콘 기반으로 재디자인**하고 서식 4종을 추가했다.

- **추가 서식**: 실행취소/되돌리기(TipTap history 내장 버튼), 형광펜(`<mark>`), 위/아래첨자(`<sup>/<sub>`, 상호배타), 블록 정렬(좌/중/우).
- **정렬은 클래스 기반**(인라인 style 금지, ADR-0021): 문단/제목에 `align` 속성 → `text-center`/`text-right` 클래스(좌측=기본, 클래스 없음). `TextAlignClass` Extension(addGlobalAttributes + setAlign).
- **형광펜/첨자는 새 의존성 없이 커스텀 마크**(`Mark.create`) — 기존 TextColorClass 패턴 따름. 도커 익명 볼륨 재빌드 함정 회피.
- **공유 화이트리스트 `richHtmlSchema`**: allowedTags += `mark`/`sup`/`sub`, allowedClasses += 정렬 클래스(p/h1/h2/h3) + h1~3 class 속성. 서버 sanitize 는 스키마 파생이라 자동 반영, `RichContent`(dompurify)도 allowedTags 파생.
- **툴바 재디자인**(`Toolbar.tsx`): 그룹+구분선. B/I/U/S·x²/x₂ 는 글자(레퍼런스와 동일), 나머지는 SVG 아이콘(`Icon.tsx` 에 undo/redo/list-bullet/list-ordered/quote/codeblock/link/rule/highlight/align-*  추가). 색/크기 select 유지. 에디터/툴바 CSS 신규(index.css).
