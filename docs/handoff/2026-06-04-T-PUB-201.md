## 결정한 것
- `UploadResultDto.type` 은 **필수**(optional 아님). 모든 응답 분기가 명시적으로 type 을 채워야 한다.
- 이름은 `UploadResultDto`(기존 컨벤션 유지). docs 의 "UploadResult" 표현은 통칭 — 코드는 Dto 접미사.
- 크기 한도는 **MIME 분리**: 이미지 5MB, 비디오 50MB. multer `limits.fileSize` 단일 정적값 한계를 컨트롤러 후검증으로 보완(T-PUB-202).

## 알려진 이슈
