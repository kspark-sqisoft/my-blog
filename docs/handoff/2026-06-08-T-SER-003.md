## 결정한 것 (ADR-0029 기반, 새 ADR 불필요)
- 멤버십·순서는 전체 목록 **원자 재지정**(부분 패치 아님). index=seriesOrder 라 중복 order 불가.
- 검증(존재·소유)과 변경을 **한 인터랙티브 트랜잭션**에 묶어 TOCTOU 경합·부분 적용을 차단(code-reviewer 권고 반영).
- 권한 경계: AUTHOR 본인 글만, ADMIN 임의(ADR-0029 #6). 시리즈 소유권은 `assertCanMutate`(소유자/ADMIN, 404/403).
- postIds 상한 100(@ArrayMaxSize) — 트랜잭션 비용 상한.
