- 회원가입을 `src/api/` 훅이 아닌 `useAuth` 스토어의 `register` 액션으로 구현.
  근거: 가입은 즉시 전역 인증 상태를 세팅(자동 로그인)해야 하고, 기존 `login`도

  (신규 ADR 불필요 — ADR-0001/ADR-0018 범위 내.)
- 폼 검증은 로컬 Zod(ADR-0004: shared는 순수 타입, 검증은 각 패키지). Login과 동일.
