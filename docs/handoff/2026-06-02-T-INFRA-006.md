## 실행 / 확인 방법

1. push 또는 PR 생성 → GitHub `Actions` 탭에서 CI 실행 확인.
2. 첫 실행은 캐시 미적중으로 5~7분 예상. 캐시 적중 후 ≤ 3분 목표.
3. 실패 시 Actions 의 해당 job 페이지에서 `playwright-report` artifact 다운로드.
4. 머지 차단을 강제하려면 저장소 settings 에서 **branch protection > required status checks**
   에 `quality`, `web-e2e` 두 체크를 추가(저장소 권한 차원이라 워크플로로는 강제 불가).
