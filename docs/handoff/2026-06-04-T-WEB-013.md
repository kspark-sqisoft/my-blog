## 결정한 것
- 인증/인가 책임 분리: `ProtectedRoute`(인증=세션 확인, 미인증→/login)는 그대로 두고,
  `RoleRoute`가 그것을 감싸 인가(역할)만 추가. 미인증→/login, 인가 실패→/ 로 분기.
  (신규 ADR 불필요 — ADR-0018 RBAC 범위 내.)
- `RoleGate`는 내부 헬퍼로 비export(파일당 1 컴포넌트 export 규칙 준수).
