# Handoff — T-WEB-308 (리스트 제목·본문 키워드 검색)

날짜: 2026-06-05 / feature: list-search

## 한 일
공개 글 목록에 제목·본문 키워드 검색을 추가했다. 입력하면 디바운스로 검색되고, 비우면 전체가 돌아온다.

- **백엔드**: `ListPostsQueryDto` + `q?`(IsString, MaxLength 100). `PostService.listPublished` where 에
  `q.trim()` 있으면 `OR: [{title contains q}, {contentMarkdown contains q}]`(대소문자 무시). 본문은 평문
  (contentMarkdown) 기준이라 HTML 태그 노이즈 없음. q 비면(공백 포함) 전체. 컨트롤러가 q 전달.
- **프론트**: `lib/useDebouncedValue.ts`(범용 디바운스 훅, 300ms) + `PostList` 검색 입력(`.ab-search`).
  검색어를 URL `?q=` 에 반영(공유/새로고침 보존), q 변경 시 page 제거(1페이지 리셋). `usePosts` 에 q 추가.
