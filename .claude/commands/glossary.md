# .claude/commands/glossary.md--

description: 유비쿼터스 언어(도메인 용어집)를 갱신한다. 새 용어 추가 또는 기존 용어 정의 검토.
argument-hint: [용어]--
docs/glossary.md 를 읽고 다음 작업을 수행:
대상 용어: $ARGUMENTS

1. 이미 정의되어 있는지 확인- 있으면 현재 정의를 보여주고 갱신이 필요한지 사용자에게 묻는다- 없으면 다음 단계로
2. 인터뷰 (한 번에 한 질문씩)- 이 용어가 가리키는 것은 무엇인가?- 비슷한 다른 용어와 어떻게 다른가? (예: Post vs Article vs Entry)- 어떤 Bounded Context에 속하는가?- 코드에서 어떤 클래스·타입·테이블로 표현되는가?- 사용자에게 어떻게 보이는가?
3. docs/glossary.md 에 다음 형식으로 추가:

## {용어} (영문 표기)- 정의: ...- 소속 Context: ...- 코드 표현: ...- UI 표현: ...- 동의어 금지: 같은 것을 가리킬 때 다른 단어 사용 금지 목록

규칙:- 한 용어 = 한 의미. 동음이의어는 명시적으로 다른 용어로 분리-
